SPlay
=====

Press a key, hear a sample.

Usage
-----

Just launch `./splay conf.json`, where `conf.json` has the following format :

    { "a": "kick.wav"
    , "z": "badamtshh.wav"
    , "e": "rimshot.mp3"
    }

The following keys are overridden :

  - `q` quits the application.
  - `m` mutes all playing clips.
